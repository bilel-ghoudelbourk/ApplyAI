# ApplyAI Frontend

This frontend is the Next.js client for ApplyAI.

Main documentation lives in the repository root:

- `../README.md`

Quick start:

```bash
cd /path/to/ApplyAI/frontend
cp .env.example .env.local
npm install
npm run dev
```

Default frontend URL:

```text
http://localhost:3000
```

Default API base URL:

```text
http://localhost:8010/api
```
